# Employee Payroll Management System

## Overview

This project is a simple Python application that calculates employee payroll details.

### Features

- Employee salary calculation
- HRA calculation
- DA calculation
- Gross salary calculation
- Tax calculation
- Net salary calculation
- Unit testing using Pytest
- Code linting using Flake8
- Continuous Integration using GitHub Actions

## Technologies

- Python 3.12
- Pytest
- Flake8
- GitHub Actions

## Run Application

```bash
python app.py
```

## Run Tests

```bash
pytest -v
```
